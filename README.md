# .NET Core Web API Projesi

Bu repo, çok katmanlı mimariye sahip bir .NET Core API projesidir.

## Tasarım Desenleri

- **Repository Design Pattern**
- **CQRS (Command Query Responsibility Segregation) Design Pattern**
- **Mediator Pattern**

## Özellikler

- Kullanıcı Kaydı
- Kullanıcı Girişi
- Token Yenileme
- Kategori Yönetimi (CRUD)
- Ürün Yönetimi (CRUD)
